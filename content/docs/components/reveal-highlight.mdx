## Usage

### Specific Section

To use the Reveal Highlight, simply wrap your app (or part of it) with the `RevealHighlightProvider`:
```tsx
import { RevealHighlightProvider } from "@/components/reveal-highlight";

export default function Page() {
  return (
    <RevealHighlightProvider>
      <div className="border p-5" />
    </RevealHighlightProvider>
  );
}
```

### Whole App

For a global effect, wrap your entire app in the provider, such as in `layout.tsx`:
```tsx
import { RevealHighlightProvider } from "@/components/reveal-highlight";

export default function Layout({ children }: { children: React.ReactNode }) {
  return (
    <RevealHighlightProvider>
      {children}
    </RevealHighlightProvider>
  );
}
```

**That's it.** Every element with a visible CSS border automatically gets the glow. No extra props, no wrappers.
